# Claude Code Skills 워크숍 v2 🎯

> **"내 일 편하게 하기"** - 비개발자를 위한 Claude Code Skills 완전 가이드

## 🚀 워크숍 개요

2시간 만에 **"내일부터 Claude Code 써야겠다!"**는 확신을 얻는 실습 중심 워크숍입니다.

### 📋 커리큘럼

**Step 1: Monday-Checklist 스킬 체험**
- 스킬 설치 → 실제 데이터 체험 → 개인 맞춤 커스터마이징
- Gmail + Slack + Calendar 자동 연동으로 차별화 확인

**Step 2: 나만의 스킬 만들기 + 팀 공유**  
- Plan Mode로 개인 반복업무 자동화 스킬 제작
- GitHub/파일 공유로 전문적 협업 체험

### 🎯 학습 목표

| 구분 | Before | After |
|------|--------|-------|
| **데이터** | 텍스트 복붙 | Gmail/Slack/Calendar 자동 연결 |
| **기능** | 질문-답변 | 워크플로우 자동화 |
| **활용** | 한 번 쓰고 끝 | 반복 사용 (매일, 매주) |

## 📦 포함된 스킬

### 1. skills-workshop
워크숍 진행을 위한 가이드 스킬
```bash
npx skills add skills-workshop
/skills-workshop
```

### 2. monday-checklist  
주말 쌓인 이메일/슬랙/캘린더를 분석해서 월요일 체크리스트 생성
```bash
npx skills add monday-checklist
월요일 체크리스트 만들어줘
```

## 🎯 시작하기

### 사전 준비
- Claude Code 설치 및 로그인
- MCP 연결 (Gmail, Slack, Calendar)
- **반복업무 1개** 미리 생각해오기

### 워크숍 시작
```bash
# 1. 워크숍 스킬 설치
npx skills add skills-workshop

# 2. 워크숍 시작
/skills-workshop
```
