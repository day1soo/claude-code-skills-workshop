# Monday Checklist - 월요일 스마트 체크리스트

## 목적
주말 동안 놓친 업무와 이번 주 일정을 한 번에 파악해서, 월요일 아침 10분 안에 한 주를 완벽하게 시작할 수 있도록 돕습니다.

## 사용법

### 기본 실행
```
월요일 체크리스트 만들어줘
```

### 커스텀 범위 설정
```
[시작날짜]부터 [종료날짜]까지 체크리스트 만들어줘
예시: 5월 10일부터 5월 16일까지 체크리스트 만들어줘
```

## 기능

### 1. 주말 이메일 분석
- 금요일 오후 6시 이후 ~ 월요일 오전 9시 이전 이메일
- 긴급도별 분류 (🔥 긴급, ⚠️ 중요, 📋 일반)
- 액션 아이템 추출 (회신 필요, 검토 필요, 승인 필요)

### 2. 슬랙 메시지 요약
- 주말 동안 받은 DM과 멘션
- 중요 채널별 주요 업데이트
- 놓친 결정사항이나 공지사항

### 3. 주간 캘린더 준비사항
- 이번 주 회의 일정과 준비사항
- 데드라인이 있는 업무
- 시간 블로킹 추천

### 4. 우선순위 체크리스트 생성
```
🌅 월요일 오전 (9-12시)
□ [긴급] 김팀장님 메일 회신 (프로젝트 승인건)
□ [중요] 화요일 회의 자료 검토
□ [일반] 주간보고 작성

🌞 월요일 오후 (13-18시)  
□ [준비] 수요일 프레젠테이션 연습
□ [검토] 신입사원 면접 일정 확인
□ [팀웍] 마케팅팀과 Q2 기획 논의

📅 이번주 주요 데드라인
• 5/14 (수) 15:00 - 분기 실적 발표
• 5/16 (금) 18:00 - 프로젝트 최종 제출
```

## 예상 효과
- **시간 절약**: 월요일 아침 혼란 30분 → 정리된 시작 5분
- **놓침 방지**: 주말 동안 쌓인 업무 100% 파악
- **우선순위**: 중요한 일부터 처리하여 효율성 극대화

## 실행 단계

### 1. MCP 도구 로드 및 연결 확인
ToolSearch로 필요한 MCP 도구들을 로드합니다:
- Gmail: `mcp__claude_ai_Gmail__search_threads`
- Slack: `mcp__claude_ai_Slack__slack_search_public_and_private`  
- Google Calendar: `mcp__claude_ai_Google_Calendar__list_events`

연결되지 않은 서비스가 있으면 연결 가이드를 제공합니다.

### 2. 시간 범위 계산
- **기본 범위**: 금요일 18:00 ~ 월요일 09:00 (주말 기간)
- **커스텀 범위**: 사용자가 날짜를 지정한 경우 해당 기간
- **시간대**: 모든 계산은 KST(한국시간) 기준

### 3. 3개 데이터 소스 병렬 수집
3명의 sub-agent를 동시에 발사하여 병렬로 데이터를 수집합니다:

**Agent 1 - Gmail 수집**:
- 주말 기간 이메일 검색
- 발신자 VIP 여부 확인
- 제목/내용 기반 긴급도 분류 (🔥긴급/⚠️중요/📋일반)
- 액션 아이템 추출 (회신 필요, 검토 필요, 승인 필요)

**Agent 2 - Slack 수집**:
- DM 메시지 수집
- 멘션(@곽연수) 메시지 수집
- 중요 채널 (#hr-team, #general) 업데이트
- 결정사항 및 공지사항 추출

**Agent 3 - Calendar 수집**:
- 이번 주 월~금 일정 조회
- 회의별 준비사항 분석
- 데드라인이 있는 업무 식별
- 시간 블로킹이 필요한 작업 추출

### 4. 데이터 통합 및 우선순위 분석
- 수집된 모든 데이터를 통합
- 우선순위 점수 계산 (긴급도 40% + 시간제약 30% + 액션필요성 20% + VIP발신자 10%)
- TOP 3 우선순위 작업 선별
- 시간대별 권장 스케줄 생성

### 5. 체크리스트 생성 및 출력
- 오전/오후별 작업 분류
- 이번주 주요 데드라인 정리
- 실행 가능한 체크리스트 형식으로 출력
- 예상 소요시간 표시

## MCP 연결 가이드

### 연결 상태 확인
```
/mcp
```
위 명령어로 현재 연결된 서비스를 확인할 수 있습니다.

### Connector 방식으로 연결 (권장)
1. **Gmail 연결**:
   - https://claude.ai/settings/connectors 접속
   - Gmail 선택하여 연결
   - Google 계정으로 로그인 후 권한 허용

2. **Slack 연결**:
   - https://claude.ai/settings/connectors 접속  
   - Slack 선택하여 연결
   - 워크스페이스 선택 (회사 정책으로 제한시 개인 워크스페이스 사용)

3. **Google Calendar 연결**:
   - https://claude.ai/settings/connectors 접속
   - Google Calendar 선택하여 연결
   - Gmail과 같은 Google 계정 사용

### CLI 방식 (백업)
연결이 안 되는 경우 CLI 방식을 시도해볼 수 있습니다:
```bash
claude mcp add gmail-server
claude mcp add slack-server
claude mcp add google-calendar-server
```
