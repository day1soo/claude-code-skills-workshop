# Step 1 — Monday-Checklist 스킬 체험

**목표**: Monday-Checklist 스킬을 설치하고 실제 데이터로 체험한 후, 개인 맞춤 버전으로 개선하기

---

## Step 1-1. 스킬 설치 및 MCP 연결

### EXPLAIN

#### Monday-Checklist 스킬이란?
**주말 동안 쌓인 이메일, 슬랙, 캘린더를 자동으로 분석해서 월요일 아침 체크리스트를 만드는 스킬**입니다.

**차별화 포인트**:
- 일반 AI: "텍스트 요약해줘" → 텍스트만 처리
- **Monday-Checklist**: Gmail + Slack + Calendar **실제 데이터 자동 수집** → 통합 분석

#### 왜 이 스킬을 선택했나?
- **즉시 활용**: 내일부터 매주 월요일마다 바로 사용 가능
- **차별화 명확**: "이건 Claude Code에서만 가능하네" 체감
- **실무 연결**: 실제 업무 데이터로 개인 맞춤 인사이트 제공

### EXECUTE

**1단계**: Monday-Checklist 스킬 설치
```bash
npx skills add monday-checklist
```

**2단계**: MCP 연결 상태 확인
```bash
/mcp
```

**3단계**: 미연결 서비스가 있으면 간단 설정
- **Gmail**: claude.ai/settings/connectors에서 간단 연결
- **Slack**: 개인 workspace 연결 (회사 제한 시)
- **Calendar**: 기본 Google 계정 연결

## Step 1-2. 실제 데이터로 체험하기

### EXPLAIN

**체험 방식**: 가짜 예시가 아닌 **본인의 실제 Gmail, Slack, Calendar 데이터**로 체험합니다.

**무엇을 확인하나?**
- 주말 이메일 자동 수집 및 긴급도 분류
- 슬랙 DM/멘션 메시지 요약
- 이번 주 캘린더 일정 및 준비사항 분석
- 우선순위 TOP 3 자동 선별

### EXECUTE

**1단계**: Monday-Checklist 실행
```
월요일 체크리스트 만들어줘
```

**2단계**: 생성 과정 관찰
- MCP 도구 로드 과정
- 3개 데이터 소스 병렬 수집 (Gmail/Slack/Calendar)
- 우선순위 분석 및 체크리스트 생성

**3단계**: 결과 분석
- **정확성**: 수집된 데이터가 실제와 일치하나?
- **유용성**: 우선순위 분류가 적절한가?
- **완성도**: 바로 업무에 활용할 수 있나?
- **차별화**: 다른 AI 도구로는 불가능한 기능인가?

## Step 1-3. 개인 맞춤 커스터마이징

### EXPLAIN

**커스터마이징이란?** Monday-Checklist를 본인의 업무 스타일과 용어에 맞게 개선하는 것입니다.

**3가지 커스터마이징 옵션**:
1. **용어 개인화** - "면접"→"인터뷰", "검토"→"리뷰"
2. **Slack DM 자동 전송** - 체크리스트를 본인에게 슬랙 DM으로 발송
3. **수집 기간 조정** - 목요일~월요일 3일간, 또는 금요일 저녁~월요일 아침

### EXECUTE

**1단계**: 개인 버전 생성
```bash
npx skills copy monday-checklist my-monday-checklist
```

**2단계**: 커스터마이징 옵션 선택
방금 체험한 결과에서 어떤 부분을 개선하고 싶은지 선택:

**A. 용어 개인화** (추천: 첫 체험자)
```
my-monday-checklist 스킬을 수정해줘.
"면접"을 "인터뷰"로, "검토"를 "리뷰"로, "회의"를 "미팅"으로 바꿔줘.
```

**B. Slack DM 자동 전송** (추천: 모바일 활용자)
```
my-monday-checklist 스킬을 수정해줘.
체크리스트 생성 후 나한테(U07ETBY353Q) 슬랙 DM으로 자동 전송하게 해줘.
```

**C. 수집 기간 조정** (추천: 업무 패턴 맞춤)
```
my-monday-checklist 스킬을 수정해줘.
수집 기간을 목요일 18:00부터 월요일 09:00까지 3일간으로 확장해줘.
```

**3단계**: 개선된 버전으로 재테스트
```
my-monday-checklist
```
