# Phase 2 Step 2 — Slack 공유 (10분) — **사이클 완주**

**목표**: 4단계 사이클의 마지막 — 내 스킬을 **공개 공간**에 올리는 경험. 이게 있어야 사이클이 몸에 박힘.

---

## EXPLAIN

### 왜 공유까지 가야 하는가

만들기만 하고 혼자 쓰면 **3일 후 잊어버려요.** 공유하면:
1. 누가 반응해 주면 → 또 만들고 싶어짐 (동기 강화)
2. 누가 내 스킬 쓴다 → 내가 다른 사람 스킬도 쓰게 됨 (사이클 순환)
3. 다음 기수 참가자가 내 스킬 가공 → **사내 스킬 카탈로그 자생**

개발자들의 오픈소스 문화가 왜 강력한지 — 공유가 디폴트이기 때문이에요.

### Slack `#claude-skills` 채널 3원칙

1. **완벽하지 않아도 올린다** (v0.1도 OK)
2. **3종 세트로 올린다** (제목 + 설명 + 파일)
3. **반응 이모지 달기는 운영 책임** — 첫 공유자에게 🙌 필수

📂 설치 방법: `npx skills add my-[스킬이름]` 또는 파일 다운로드

Claude가 본인 스킬 description과 body를 읽고 자동으로 채워서 보여줌. 수정 필요하면 Edit으로.

### Step 2-2. Slack 업로드 (5분)

**직접 작업**:

1. 방금 저장한 `C:\Users\admin\.claude\skills\my-[업무이름]\SKILL.md` 파일을 **Slack에서 `#claude-skills` 채널** 열기
2. 위 공유 메시지 텍스트를 채널에 붙여넣기
3. `SKILL.md` 파일을 메시지 박스에 **드래그 & 드롭** (또는 클립 아이콘)
4. "보내기"

**운영 약속** (강사가 사전 고지):
- 게시 30분 안에 누군가 🙌 이모지 달아줌
- 하루 안에 한 명 이상 댓글 피드백
